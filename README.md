# Calculadora Venezuela

App estático modular para situações comuns de compra na Venezuela:

- Comparar pagamento por Zelle/USD direto contra pagamento em bolívares usando taxa de loja, taxa real de câmbio e preço de referência em dólar ou euro.
- Converter preço em bolívares para USD usando a taxa escolhida, USD BCV ou USD paralelo.

Publicação via GitHub Pages:
